Abstract

Effective robot manipulation requires translating high-level intent into precise physical interaction. A common approach is to use language as a task specification, with vision-language models providing a flexible interface for interpreting instructions and reasoning about the scene. However, reliable execution often requires more than semantic task understanding: manipulation policies benefit from explicit geometric goals that specify where and how the robot should interact with the environment. We introduce Point2Skill, a framework for specifying reusable manipulation skills through 3D target points. Each primitive is modeled as a Diffusion Transformer policy conditioned on a goal point via adaptive Layer Normalization, allowing geometric intent to modulate the denoising process with out the overhead of cross-attention. We instantiate a library of primitives, including Pick, Place, Insert, Open, and Close, and provide an interactive click-based interface through which either a human or a Vision-Language Model can select. Do not distribute. the target point. This point-based parameterization abstracts skills away from object identity, enabling a single primitive to generalize across object categories and novel instances. By separating task-level reasoning from motion generation, the proposed framework supports reuse and composition of manipulation skills. We further show that Vision-Language Models can sequence goal-conditioned primitives to perform long-horizon manipulation tasks. Experiments on a Franka robot across multiple tasks and objects demonstrate strong generalization to unseen objects while enabling reliable execution of composed behaviors

Primitive Skills

Point2Skill learns a library of reusable goal-conditioned primitives. Each primitive receives visual observations, robot proprioception, and a user- or VLM-selected 3D goal point, and outputs an action trajectory through a goal-conditioned Diffusion Transformer.

Pick

A 3D goal point specifies where the robot should grasp.

Place

The goal point defines the target placement location.

Insert

The policy aligns the object with a constrained receptacle.

Open / Close

The goal point anchors interaction around the handle.

Shared Goal-Conditioned Diffusion Policy

All primitives share the same policy structure. Multi-view wrist observations and proprioception are encoded together with a 3D goal point. The goal embedding is combined with the diffusion timestep and injected into the Diffusion Transformer through adaLN, allowing the target interaction point to modulate the denoising process.

01

Multi-view observations

Fisheye RGB and wrist RGB-D images provide scene context and local geometry.

02

3D goal point

A user or VLM click is lifted to 3D and used as the geometric target.

03

adaLN conditioning

The goal and timestep embeddings modulate the DiT denoising blocks.

04

Action trajectory

The policy predicts a sequence of robot actions for the selected primitive.
