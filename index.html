Abstract

Effective robot manipulation requires translating high-level intent into precise physical interaction. A common approach is to use language as a task specification, with vision-language models providing a flexible interface for interpreting instructions and reasoning about the scene. However, reliable execution often requires more than semantic task understanding: manipulation policies benefit from explicit geometric goals that specify where and how the robot should interact with the environment. We introduce Point2Skill, a framework for specifying reusable manipulation skills through 3D target points.
